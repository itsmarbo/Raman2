In this section you will find all the recquirements to obtain the 
hessian from an h5 output.


  FILE		RESULT		OBJECTIVE
h5ssian.py	hessian.txt	parsing to obtain HESSIAN from .h5
parssian.py	filename.fcc	parsing to obtain FCC input


It is important to note that, to obtain the hessian, the CASPT2 calculation 
must have specified that in the input file (Open Molcas). Otherway the 
hessian is not going to be provided.
